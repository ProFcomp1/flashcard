Variável

Armazena um valor para ser usado no programa.

Exemplo:

let nome = "Maria";

Cria uma variável chamada nome.

Função

Bloco de código reutilizável que realiza uma tarefa.

Exemplo:

function somar(a, b) { return a + b; }

Função que retorna a soma de dois números.

Condicional

Permite executar ações diferentes com base em uma condição.

Exemplo:

if (idade >= 18) { alert("Maior de idade"); }

Verifica se a idade é 18 ou mais.

Loop

Repete um bloco de código enquanto uma condição for verdadeira.

Exemplo:

for (let i = 0; i < 5; i++) { console.log(i); }

Imprime os números de 0 a 4 no console.

Flashcard CSS – Seletor

Seletor CSS

Define quais elementos HTML receberão estilos.

Exemplo

p { color: blue; }

Aplica cor azul a todos os parágrafos (<p>).
